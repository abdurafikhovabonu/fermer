# 🌾 Farmers School Management System

A web-based platform designed to manage farmer education processes, track activities, and simplify administration tasks through an intuitive admin panel.

## 🚀 Features

- 👨‍🌾 Farmer registration and management
- 📊 Admin dashboard for monitoring data
- 📦 Product / activity tracking
- 🔍 Search and filtering system
- 📱 QR code integration (if used)
- 📈 Reports and statistics
- 🔐 Authentication (login system)

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- JSON file-based storage (local database)
